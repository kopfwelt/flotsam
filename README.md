# Kern

## Introduction
Kern is a CSS framework based on [SUIT CSS](https://suitcss.github.io) by [KOPFWELT](https://kopfwelt.com).

## Install

```sh
$ npm i -S kopfwelt/kern
```
## Test

```sh
$ npm test
```

Watching

```sh
$ npm test -- --watch
```
## Generating dockset



```sh
$ npm run dockset
```
